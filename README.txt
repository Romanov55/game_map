Тестовое задание 'игровая карта' на чистом js

1. При клике на кнопку 'В Универ' осуществляется передвижение персонажа с анимацией по карте на 1 секцию вперёд.
2. Слайдер ленты контактов
3. При клике на кнопку 'Рейтинг' открывается модальное окно с рейтингом игроков с данными из файла data.js, объекта 'rating' и выделением в списке друзей из объекта 'friends' из того же файла
Добавлены небольшие анимационные действия.

Было потрачено 10 часов времени, некоторая сложность была с отрисовкой анимации движения персонажа по точкам вдоль дорожки. (около 2 часов)
